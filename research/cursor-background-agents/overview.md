### Core Functionality
- **Asynchronous Operation**: Agents run independently in remote environments ‚Üí [Technical Architecture](./technical-architecture.md#remote-environment)

- **Real-time Monitoring**: Status updates and intervention capabilities ‚Üí [Access & Control](./access-control.md#user-interface)

### Technical Implementation  
- **Environment Setup**: Ubuntu-based isolated machines ‚Üí [Technical Architecture](./technical-architecture.md#environment-setup)

- **Security**: Privacy Mode with encrypted secrets ‚Üí [Security & Privacy](./security-privacy.md#privacy-mode)
- **Performance**: Disk state caching for fast startup ‚Üí [Operational Considerations](./operational-considerations.md#performance)
